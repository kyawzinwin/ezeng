# EzEng

EzEng is a simple flashcard app for Burmese speakers who want to learn English.

It gives you a deck of English **words, phrases, and idioms**, each paired with
its Burmese meaning. Tap a card to flip between English and Burmese, and switch
which language shows first so you can practice in either direction. Words also
include pronunciation (IPA + audio) to help you say them correctly.

The deck is shuffled every time you open the app, so you never practice in the
same order twice.
